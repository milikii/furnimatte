# 家具自动抠图工具 (furniture_cutout)

极简 Windows 桌面工具。打开实拍家具照片 → 自动识别主体去除背景 → 输出透明 PNG。

纯本地运行，**不上传图片、不联网**（首次模型下载除外）。

## 环境要求

- Windows 10/11 64-bit
- Python 3.11 64-bit
- Intel Core i5 或同级 CPU（建议 ≥6 核）
- 32 GB 内存（16GB 处理 4K 照片可能紧张）
- **无需独立显卡**（纯 CPU 运行）

## 安装

1. 安装 [Python 3.11 64-bit](https://www.python.org/downloads/)（安装时勾选 "Add Python to PATH"）。
2. 双击 `install.bat`，等待自动完成：
   - 创建虚拟环境 `.venv`
   - 安装 CPU 版 PyTorch（`--index-url https://download.pytorch.org/whl/cpu`）
   - 安装其他依赖
3. 看到 `安装完成` 即成功。

> 国内用户若模型下载超时，可在设置中开启 **HF 镜像**（`hf-mirror.com`），或手动选择已下载的本地模型目录。

## 启动

双击 `start.bat`。首次启动后点击"自动抠图"会下载模型（~800MB），请耐心等待。

## 使用流程

### 标准流程：打开 → 抠图 → 保存
1. 点击 **打开图片**（或拖入照片）→ 支持 `.jpg/.jpeg/.png/.webp`
2. 点击 **自动抠图** → 等待推理完成（CPU 推理约 30s–2min，属正常）
3. 在右侧查看透明结果（可切换棋盘格/白色/黑色背景检查边缘）
4. 点击 **保存 PNG** → 输出 `<原名>_cutout.png`，存原图目录

### 框选重算（自动结果不准确时）
1. 点击 **框选重算**
2. 在左侧原图上**鼠标拖动框住正确家具**
3. 自动对框选区域重新抠图，框外全透明
4. 满意后保存

> 框选区域小于 50×50 像素时会提示重新框选。

### 其他按钮
- **取消框选** — 退出框选模式
- **重置** — 回到刚打开图片的状态
- **设置** — 调整框选扩展比例、CPU 线程数、输出目录等

- 可在设置中手动指定已下载的本地模型目录

## 常见问题

**Q: 模型下载失败？**
A: 国内网络——在设置中开启 **HF 镜像**；或者手动下载模型后，设置中指定模型目录。

**Q: CPU 推理很慢？**
A: 正常。BiRefNet_HR-matting 在 CPU/float32 下处理一张 2048² 图片约需 1–3 分钟。质量优先，不牺牲精度换速度。

**Q: 内存不足？**
A: 关闭其他程序；或缩小图片尺寸（先在外压缩再打开）。

**Q: 程序出错？**
A: 查看 `logs/app.log` 获取详细错误信息。

**Q: 如何卸载？**
A: 删除整个项目文件夹即可（模型缓存默认在 HuggingFace 缓存目录，可一并删除）。
